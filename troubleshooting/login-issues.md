# Login Issues Troubleshooting Guide

## Problem

A customer reports they cannot log in to their account.

---

## Information to Gather

- User's email address
- Exact error message
- Browser and version
- Device and operating system
- Time the issue started
- Whether the issue affects only one user or multiple users

---

## Investigation Steps

### 1. Verify the account exists.

### 2. Check if the account is active.

### 3. Confirm the user is entering the correct email address.

### 4. Check whether the password has expired.

### 5. Ask the user to reset their password.

### 6. Test using an Incognito/Private browser window.

### 7. Clear browser cache and cookies.

### 8. Verify internet connectivity.

### 9. Check the system status page for outages.

---

## Possible Causes

- Incorrect password
- Expired password
- Locked account
- Browser cache issue
- Network issue
- Authentication service outage

---

## Resolution

Resolve the identified issue or escalate to the engineering team if the problem is caused by a backend service.
