# Login Issue Troubleshooting Guide

## Issue

A user is unable to log in to the application.

---

## Symptoms

- Incorrect username or password error
- Blank login page
- Multi-factor authentication code not received
- Session expires immediately after login

---

## Possible Causes

- Invalid credentials
- Expired password
- Browser cache or cookies
- Network connectivity issues
- Account locked after multiple failed attempts
- Authentication service outage

---

## Troubleshooting Steps

1. Verify the user's email or username.
2. Reset the password if necessary.
3. Clear browser cache and cookies.
4. Try logging in using an incognito/private window.
5. Test using another browser or device.
6. Check the service status page for outages.
7. Review authentication logs (if available).
8. Escalate if the issue cannot be resolved.

---

## Expected Resolution

The user successfully logs in after correcting credentials or resolving the underlying issue.

---

## Escalation Criteria

Escalate the issue if:

- Multiple users report the same problem.
- Authentication service is unavailable.
- The issue persists after completing all troubleshooting steps.
